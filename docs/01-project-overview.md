# Project Overview

## What is STRIDE?
STRIDE is a mobile application built with React Native and Expo, designed for planning and tracking everyday tasks. It allows users to set priorities, schedule deadlines, and organize recurring activities effectively.

## Key Features
- **Task Management**: Create, edit, and delete tasks.
- **Prioritization**: Categorize tasks by High, Medium, or Low priority.
- **Advanced Scheduling**: Set specific dates and times. Configure recurring events such as daily, weekdays, weekly, monthly, or yearly tasks.
- **Weekly Overview**: A dashboard displaying day cards with progress bars showing the ratio of completed to total tasks.
- **Day Detail View**: A focused list of active and completed tasks for a specific day, complete with status toggles.
- **Swipe Actions**: Intuitive UX allowing users to edit or delete tasks using swipe gestures.
- **Calendar Navigation**: Easily switch between weeks using a built-in calendar modal.
- **Secure Authentication**: Email and password authentication powered by Firebase Auth.
- **Cloud Sync**: All user data is securely stored and synced in real-time using Firebase Firestore.
- **Multi-language Support**: Automatically detects device locale and supports English, German, and Russian.
- **In-App Notifications**: Toast messages for feedback on user actions (e.g., successful login, task creation).
